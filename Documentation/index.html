Installation

Main zip file HTML and Documentation Included

Phoenix-HTML-Package.zip file extract and main HTML template and documentation folder are included.

(1) Phoenix-HTML and (2) Documentation folder

Open the "Phoenix-HTML" folder and run index.html in your browser

  1. Open the Phoenix-HTML Folder to find all the Templates Files
  2. You will need to Upload these files to your Web Server using FTP in order to use it on your Website.
  3. Make sure you upload the required files/folders listed below:
    • Phoenix-HTML/assets/css - Extra Stylesheets Folder
    • Phoenix-HTML/assets/fonts - Fonts Folder
    • Phoenix-HTML/assets/images - Images Folder
    • Phoenix-HTML/assets/js - Javacripts Folder
    • Phoenix-HTML/index.html - Index File/Homepage
    The other files can be used according to your preferences.
  4. You're now good to go..! Start adding your Content and show off your Brand New Beautiful Website in style.

FAQ



How to remove the top menu sticky as scrolling?

  • Open Up your HTML file 'Phoenix-HTML/' file location. (For Example, 'index.html'
  • Then Search For the Comment '<!-- START HEADER -->'
  • show this line - <header class="header_wrap">
  • add class name - <header class="header_wrap no-sticky"> (add class name 'no-sticky')

Where i can define my e-mail account to receive the messages from contact form?

  • Open Up your HTML file 'Phoenix-HTML' file location. (For Example, 'Phoenix-HTML/contact.php'
  • show this line number 11 - $mailTo = 'email@example.com'
  • 'email@example.com' enter your email account to receive the messages from contact form
